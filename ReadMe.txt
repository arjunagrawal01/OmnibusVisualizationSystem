Installation

1. Unzip the Project.zip file.

2. All code files are located in the subfolder Project_Files.

3. Run the MainMenu.html file

Running the Program

1. The main page has 6 menus. The charts which involves one variable is aligned vertically at the left side of the main page 
and the charts which involves multiple variables is aligned at the right side.

2. Each subpage has two buttons, which are Draw Diagram and Main Menu. Put JSON code to the left side of each corresponding web page.

3. After making the JSON input, Draw Diagram shows various types of charts depending on the menu the user chooses. Main Menu button located at the bottom simply redirects the user

to the main page. The user can either choose different menu or just exit.
